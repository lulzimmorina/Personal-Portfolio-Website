## Project Description
This project is ultimately way more than a simple personal portfolio. I created it for one goal and one goal only: to ensure that I, as an individual, understand the ins and outs of our digital world. 

My vision is to develop such profound skills in Computer Science that I can understand complex systems, their usage, and finally be able to leverage these systems to not only create a career for myself but to solve real-life "quality-of-life" human problems. It all starts here, with a simple, knowledge-based website.
